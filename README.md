梯度的 norm 值我們可以設定程式停止值，並不斷搜尋解答，並透過每次計算數值的改變來
計算收斂速度以驗證效率。為了近一步提升收斂速度，藉由共厄梯度下降法來達成 n+1 的快
速收斂(n 為維度)。實際比較可以發現收斂速度由 56 次(如圖 1)提升到僅僅 4 次(如圖 2)。完
